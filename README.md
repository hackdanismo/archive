# Archive

## What is Archive?
The `Archive` is a collection of code examples, documentation and developer resources to collate knowledge in a single location for sharing. In the world of AI and LLMs, it is important to understand code and the underlying technologies behind how things work. The `Archive` is a means to do this.

All content is within `markdown` files with the file extension of `.md` and any code examples are free to use without any licence restrictions.
